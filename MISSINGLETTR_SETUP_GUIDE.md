## ‚ùì **COMMON ISSUES & SOLUTIONS**

### **Issue: Image Won't Upload**
- **Solution:** Check file size (max 5MB), try PNG instead of JPG

### **Issue: Instagram Connection Failed**
- **Solution:** Disconnect and reconnect, check password

### **Issue: Post Didn't Publish**
- **Solution:** Check scheduled time, verify account connection

### **Issue: Link Page Not Showing**
- **Solution:** Make sure page is published, not just saved as draft
