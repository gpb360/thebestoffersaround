### Adding a Product Review (2000+ words)

**File:** `/home/garyp/astro-affiliate-site/src/data/products.ts`

1. Find the product object
2. Add these fields (if not present):
```typescript
fullReviewContent: `Full HTML review content (2000+ words)`,
metaTitle: 'Product Name Review 2026: Benefits, Ingredients & Side Effects',
metaDescription: 'Comprehensive review (150-160 chars)',
keywords: ['keyword1', 'keyword2', 'keyword3']
```

3. **Review Structure Template:**
```html
<h2>What is [Product]?</h2>
```
