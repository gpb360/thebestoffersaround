## Troubleshooting

### Issue: Username Taken
**Solution:** Use alternative username from list above. Consistency across platforms is ideal but not critical.

### Issue: Instagram Business Account Requirements
**Solution:** Connect a Facebook Page. Create a personal Facebook account first if you don't have one.

### Issue: Pinterest Website Verification
**Solution:** Upload HTML file to your site or add meta tag. Already done with Astro setup.

### Issue: LinkedIn Company Page Requires Personal Profile
**Solution:** Create personal LinkedIn profile first, then create company page.
